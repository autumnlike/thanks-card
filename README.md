# Thanks Card

プロジェクト終わりにメンバー同士で「ありがとう」のカードを送り合うアプリです。

HTMLファイル1枚で完結。サーバー不要、個人情報はURLパラメータのみに含まれます。

## 使い方

### 1. メンバーを登録してURLを作成

https://autumnlike.github.io/thanks-card/ にアクセスし、メンバーの名前を1行ずつ入力して「URLを生成」ボタンを押します。

### 2. 生成されたURLをメンバーに共有

生成されたURLをSlackなどで共有します。

### 3. カードを作成

共有されたURLを開くと、カード作成画面が表示されます。

1. 送り先のメンバーを選択
2. 自分の名前を入力
3. デザインを選択（かわいい / ハートフル / ロック / ナチュラル）
4. メッセージを入力
5. プレビュー → ダウンロード

ダウンロードしたカード画像をSlackのDMで送りましょう！

## 特徴

- **HTMLファイル1枚** で完結、サーバー不要
- **個人情報なし** - メンバー名はURLクエリストリングのみ
- **4種類のデザイン** - かわいい🌸 / ハートフル🧡 / ロック🎸 / ナチュラル🌿
- **文字サイズ自動調整** - メッセージの長さに応じて最適なサイズに
- **PNG画像ダウンロード** - ファイル名は「○○から○○へ.png」形式
