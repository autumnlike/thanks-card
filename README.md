# Thanks Card

プロジェクト終わりにメンバー同士で「ありがとう」のカードを送り合うアプリです。

HTMLファイル1枚で完結。サーバー不要、個人情報はURLパラメータのみに含まれます。

## 使い方

### そのまま使う

https://autumnlike.github.io/thanks-card/ にアクセスするだけですぐ使えます（サンプルメンバーが表示されます）。

### チームで使う

1. ページ上部の「メンバーを変更 ✏️」からメンバーを入力
2. 「共有URLを生成 🔗」でURL作成 → メンバーにSlackなどで共有
3. URLを開くとチームメンバーが選択肢に表示されます

### カードを作成

1. 送り先を選ぶ（ボタン選択 or 直接入力）
2. 自分の名前を入力
3. 14種類のデザインから選択
4. メッセージを入力
5. プレビュー → ダウンロード
6. SlackのDMやメールにカード画像を貼り付けて送ろう！

## デザイン一覧

🌸 かわいい / 🧡 ハートフル / 🎸 ロック / 🌿 ナチュラル / ☕ カフェ / 🌅 サンセット / 🌊 オーシャン / 🌙 ナイトスカイ / ❄️ ウィンター / 🍭 レトロポップ / 🎪 フェスティバル / 🌈 レインボー / 🖤 モノトーン / 🎩 エレガント

## 特徴

- **HTMLファイル1枚** で完結、サーバー不要
- **個人情報なし** - メンバー名はURLクエリストリングのみ、コードに含まれない
- **14種類のデザイン** - 温かみ系からクール系、ポップ系、シック系まで
- **文字サイズ自動調整** - メッセージの長さに応じて最適なサイズに
- **PNG画像ダウンロード** - ファイル名は「○○から○○へ.png」形式
- **手動入力対応** - メンバーリストにない人にも直接名前を入力して送れる
- **アクセス解析** - GoatCounterによるプライバシー配慮型の計測
